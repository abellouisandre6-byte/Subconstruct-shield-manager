# Subconstruct-shield-manager
A plug and play script for from the depths that creates a shield wall from subvehicles.

the script has a few settings to tune depending on the size of the drones and the quantity of drones.

REQUIREMENTS

1) Subvehicle holders must be on a two axis turret
2) turrets with subvehicle holders must be facing upwards

MUST-CHANGE PARAMETERS - not doing so may induce critical errors

1) down in shield wall shape, adjust GridRows and GridColumns so that GridRows*GridColumns > # of drones

...

OTHER PARAMETERS

DroneGap --> controls the gap between drones. adjust if drones are clipping or are too far apart


You should not need to change anything else, but additional parameters are available.
